# 🎓 EduScout – Online Course Comparison Demo App

## 📌 About The Project

EduScout is a frontend demo web application that allows users to browse and compare online courses.

This is **not a full-scale startup product**, but a practice/demo project built to demonstrate frontend development skills such as DOM manipulation, filtering logic, cart handling, and UI rendering using JavaScript.

The goal of this project is to simulate how a course comparison platform might work — purely on the frontend side.

---

## 🚀 What This Project Demonstrates

- Dynamic rendering of course cards
- Category-based filtering
- Sorting by rating, price, and duration
- Add to Cart functionality
- Course comparison table
- Simulated checkout process
- Responsive UI using Tailwind CSS classes
- DOM manipulation using Vanilla JavaScript

---

## 🛠 Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (Vanilla JS)

No backend, database, or real payment integration is included.
All data is mock data stored inside the JavaScript file.

---

## 📂 Project Structure

EduScout contains only two files:

- `eduscout.html` → Contains the full UI structure
- `script.js` → Contains all rendering logic, filtering, cart handling, and navigation logic

There is no separate CSS file. Styling is handled using Tailwind CSS utility classes inside the HTML file.

---

## ▶ How To Run

1. Download or clone the repository
2. Open `eduscout.html` in your browser

No installation or setup required.

---

## ⚠ Important Note

This project is created for learning and demonstration purposes only.

- The checkout process is simulated.
- There is no real payment gateway.
- There is no backend server.
- All course data is static mock data.

The purpose is to showcase frontend logic and UI behavior — not to represent a production-ready platform.
